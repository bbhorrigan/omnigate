

# Security Policy – Omnigate

## Supported Versions

The following Omnigate versions receive active security updates and vulnerability patches. Older versions may still receive critical security fixes on a best-effort basis, but are otherwise considered unsupported.

| Version | Supported |
| ------- | --------- |
| 1.2.x   | Yes       |
| 1.1.x   | Yes       |
| 1.0.x   | No        |
| < 1.0   | No        |

---

* Subject line: “Omnigate Security Report – \[Short Summary]”

**What to include:**

* A clear description of the issue
* Steps to reproduce (if possible)
* Any logs, proof-of-concept code, or screenshots
* Your contact information for follow-up

**Response expectations:**

* Initial acknowledgment within 48 hours
* Status updates every 5 business days until resolution
* If confirmed, coordinated disclosure will be agreed upon before public release
* Security advisories will be published at github

**Scope:**
This policy covers Omnigate source code, container images, Helm charts, Terraform modules, and Omnigate-managed infrastructure.
